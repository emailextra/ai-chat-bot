# Scope Statement – AI-Powered Chatbot for Customer Support

## Project Summary

| Item | Details |
|---|---|
| Project title | AI-Powered Chatbot for Customer Support |
| Project purpose | To create a website chatbot that answers common customer questions and transfers complex cases to human staff. |
| Main users | Customers, customer support staff, project manager, business owner and technical team. |
| Project duration | 8 weeks |
| Project method | Planned project approach with weekly tracking and controlled changes. |

## In Scope
- Collecting customer support requirements and common FAQs.
- Designing chatbot conversation flows for common customer questions.
- Creating a basic knowledge base for approved answers.
- Developing the chatbot logic and website chat widget.
- Adding human handover for questions the chatbot cannot answer.
- Testing chatbot accuracy, usability and basic security controls.
- Preparing a short user guide and handover note for support staff.
- Deploying the chatbot as a first release on the sample website.

## Out of Scope
- Replacing the full customer service department.
- Voice-based chatbot or telephone support automation.
- Mobile application development.
- Payment processing through the chatbot.
- Full CRM integration in the first release.
- Multilingual chatbot support unless approved through change request.

## Acceptance Criteria
- The chatbot can answer the approved FAQ questions with clear and consistent responses.
- The chatbot has a working human handover option for unresolved questions.
- The chatbot is connected to the website and can be used from the customer page.
- Testing has been completed and major defects have been fixed before go-live.
- A simple support staff handover guide has been prepared.
- The final project document includes scope, Gantt schedule, risks, issues and change process.
